# Installation

1. Download and unzip the plugin to a permanent location on your computer.
2. Open Chrome and navigate to:
   ```
   chrome://extensions
   ```
3. Enable **Developer mode** (top-right corner).
4. Click **Load unpacked**.
5. Select the **`orion-tv-plugin`** folder.
6. Reload the **Orion Screener** page.

## You're Done! 🎉

The TradingView & Bitunix icon should now appear in the screener. Clicking the icon will open the selected symbol's TradingView chart in a new browser tab.
